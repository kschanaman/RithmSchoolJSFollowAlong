Arrays2: Using Splice to Modify Arrays

In the previous exercise (tut09-arrays.html) we learned how to use various array functions to add, remove, and modify elements:

  • push() - Add an element to the end of an array (returns length of array)
  • pop() - Remove an element from the end of an array (returns element)
  • unshift() - Add an element to the beginning of an array (returns length of array)
  • shift() - Remove an element from beginning of array (returns element)

Splice is used in the following way:

arrayName.splice(a,b,c);

Where 'a' is the element of the array where you want to start the operation, 'b' is the number of elements to remove, and 'c' (and others) are the elements you want to add beginning at the location signified by 'a'. If you don't want to remove anything, let 'b' = 0.
