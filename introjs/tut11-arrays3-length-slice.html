Arrays3: Playing With Length and Slice in Arrays

We can use the length property (it isn't a function) to determine how many elements exist in an array. This can be handy when we come to the point where we need to know the maximum number of iterations to make covering an entire array via a loop, so we don't exceed that number. If we exceed the number of elements in an array, we will get 'undefined' results for all non-existent elements.

The slice() function, used by itself, will make a copy of an entire array. This is placed into a new variable for future use. But slice also takes two arguments much like splice() covered in the last example ( tut10-arrays2-splice.html). The first number is the first element where you want to begin slicing. The second number is the last element where you want to stop slicing (but that last element is NOT INCLUDED IN the slice - key point to remember!!).
