Using Forms Instead of Prompt Boxes

Instead of continuing to use prompt() popups while learning JavaScript, I felt it is now time to re-learn how to use forms for input, and using JavaScript to read and process those items. After all, we ARE using a language meant for doing web page stuff, correct?

At the time of this writing, it has been many years since I last did any web developement. I was good at it at the time (2000 - 2004) but then life got in the way with jobs and family and I had to put things down for a long period of time (it's 2017 -- almost 2018 now). I got very rusty with forms, and so here I am starting over. Not a bad thing since HTML5 has added much to forms these days anyway, which must be learned.
