For Loops: Iteration using a FOR Loop

A 'for' loop in JavaScript appears as it does in C language for the most part. You can declare a var within a for loop, or you can create it at the top of the function in which the loop exists (or globally if desired, though global scope must be dealt with using caution, as with any other language).

A 'for' loop looks as follows:

for (initializer; condition; counter) {}

  • initializer: declared variable to hold starting number, to be used in the loop
  • condition: this MUST be an expression that returns true or false. It can be read as "Keep looping as long as this condition is true."
  • counter: this is how we change the variables initialized (either by increasing or decreasing them). Commonly, we increment variables by 1 using the ++ operator and decrement by 1 using --.

As long as the condition is true, the code inside the curly braces will run. After running, the counter expression will run, and then the condition will be checked again.
