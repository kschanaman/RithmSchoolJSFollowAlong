Arrays: Creating, Changing, Push/Pop, Shift/Unshift

Arrays are quite useful for holding lists of items in a sequential container. An array is created by creating a variable to contain it so it is accessible in the program. The elements go within brackets. An array looks like this:

var array = [1, 2, 3, 4];

Array elements are counted from zero. So in the above, 1 is inside of element 0, 2 is inside of element 1, 3 is inside of element 2, and 4 is inside of element 3.
