Using Functions as Calculator

Carrying on with functions, we now play with sending arguments to them to be used in doing something somewhat useful. The goal of this small application is to allow the visitor to enter two numbers, and to select what operation is to be performed with those two numbers: add, subtract, multiply, or divide. The operation will be selected by clicking on the necessary radio button. The necessary function in the JavaScript code file for this page will then be called. The function will perform the calculation and send the result to the desired part of the page below.
