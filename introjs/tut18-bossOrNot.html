Tutorial 18: Boss? Or Not?

This little exercise is the second one on Rithm School's free introduction to JavaScript, under the "Functions and Parameters" section of the short course. The exercise asks the student to write a function named 'sayHello' which takes in a string as a parameter. Built into the JavaScript code for this exercise is a variable called "bossName" where the name of the boss (the student) is stored. The function should have a special greeting if that is the name that is passed. Otherwise, it should simply say hello to anyone else, including their name.
