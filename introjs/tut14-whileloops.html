Using 'while' Loops in JavaScript

While loops are similar to for loops, with the exception that you must manually add an increment or decrement statement within the loop, and your counter must be initialized before being used in that while loop.

In this page, you will be asked for two numbers which will be added together. A while loop will check to make sure you entered a number. As long as the input of either entry is NOT a number, the while loop will print an error message to this HTML page telling you one of the entries wasn't a valid number. Once you've entered two actual numbers, the while loop will break out and have the program do the addition and output for you.
